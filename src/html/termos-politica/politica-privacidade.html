Política de Privacidade

Como protegemos seus dados

Introdução

A Find Invest está comprometida em proteger sua privacidade e seus dados pessoais. Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos suas informações.

Dados que Coletamos

  • Dados Cadastrais

    Nome, e-mail, telefone e documentos necessários para identificação.

  • Dados de Investimento

    Histórico de investimentos, perfil de investidor e preferências.

  • Dados de Uso

    Informações sobre como você utiliza nossa plataforma.

Como Usamos seus Dados

Segurança

Proteção contra fraudes e verificação de identidade.

Serviços

Fornecimento e melhoria dos nossos serviços.

Personalização

Recomendações personalizadas de investimentos.

Seus Direitos

  • Acesso aos Dados

    Você pode solicitar acesso a todos os seus dados pessoais.

  • Correção

    Correção de dados incompletos ou inexatos.

  • Exclusão

    Solicitar a exclusão dos seus dados pessoais.

Segurança dos Dados

Implementamos medidas técnicas e organizacionais apropriadas para proteger seus dados pessoais contra processamento não autorizado ou ilegal, perda acidental, destruição ou danos.

Criptografia

Dados sensíveis são criptografados.

Servidores Seguros

Armazenamento em servidores protegidos.

Controle de Acesso

Acesso restrito a dados pessoais.
